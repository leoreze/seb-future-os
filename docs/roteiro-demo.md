# Roteiro de demonstração — SEB Future OS v1

"O SEB Future OS é uma plataforma proprietária de inteligência educacional 360. Ele conecta rede, escola, coordenação, professores, estudantes e famílias em uma jornada única, usando dados e IA para transformar sinais dispersos em ações concretas."

Entrar como `rede@sebfuture.local` com senha `123456`.
